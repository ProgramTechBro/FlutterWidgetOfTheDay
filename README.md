## Introduction

Flutter is a UI toolkit for building natively compiled applications for mobile, web, and desktop from a single codebase. The goal of this project is to enrich your Flutter knowledge by exploring different widgets each day and implementing them into practical examples.

## Project Structure

The project is structured as follows:
```
FlutterWidgetOfTheDay/
├── lib/
│   ├── widgets/
│   ├── models/
│   ├── utils/
│   └── main.dart
├── test/
├── pubspec.yaml
└── README.md
```
- **lib/**: Contains the main application code.
- **widgets/**: Contains the implementation of various Flutter widgets.
- **models/**: Data models used in the application.
- **utils/**: Utility functions and classes.
- **main.dart**: The entry point of the application.

## How to Run

To run the Flutter Widget Of The Day application, follow these steps:
1. Clone this repository:
   ```bash
   git clone https://github.com/ProgramTechBro/FlutterWidgetOfTheDay.git
   ```
2. Navigate to the project directory:
   ```bash
   cd FlutterWidgetOfTheDay
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the application:
   ```bash
   flutter run
   ```
